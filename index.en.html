Text Editor "SAKURA"

"SAKURA" is a Japanese text editor for MS Windows.

Ver. 1.x.x.x(ANSI version) Japanese MS Windows 95/98/NT/2000/XP/Vista/7/8/10 is required.
Most double-byte characters are not shown properly on other language versions of MS Windows.

Ver. 2.x.x.x(Unicode version) MS Windows 2000/XP/Vista/7/8/10 is required.
English User Interface is supported since 2.1.1.1

Most pages linked below are written in Japanese. A Japanese font should be installed on your PC to enjoy them.
