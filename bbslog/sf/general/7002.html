      ＞例えば 4 TABで、変換すると余分なTABが入ります。
      余分というのは、余計なSPACE->TAB変換という意味です。
      ヘルプには保証動作が書いてありますから、現状の動きはヘルプの通りです。
      目標としては、
      int a;
      int a;
      int a;
      int a;
      int TABa;
      現状は半角スペース１個がタブ１個に変換されています。
