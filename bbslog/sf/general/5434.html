    • [5436] Re:行頭・行末のカーソル移動 maru 2006年09月21日 23:55

      ▼ さわださん
      ・行頭で左を押しても1行上の行末に移動しない
      ・行末で右を押しても次の行の行頭に移動しない
      が期待の動作でしょうか。

      // カーソルキー左に割り当てるマクロ
      if S_ExpandParameter('$x') <> '1' then S_Left;

      // カーソルキー右に割り当てるマクロ
      S_Right;
      if S_ExpandParameter('$x') = '1' then S_Left;

      ※フリーカーソルモードは考慮していません。
