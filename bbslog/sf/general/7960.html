  • [7960] 複数ある指定拡張子の設定を一括でしたい 質問 2015年02月23日 07:56

    環境 windowsXP vista 7 8.1
    バージョン2.1.1.4 (今2.0.2.0から上書きしたばかり)

    当方、多くのテキストファイルを扱い、
    いろいろな拡張子を自分で用途に応じて付けて
    sakuraで編集しています。
    多くの中古PCとOSでsakuraのフォルダをコピーしてsakuraを使用しています。

    キーワードの色付はタイプ別設定で行うようですが、
    タブの 「スクリーン」 「カラー」 「正規表現キーワード」
    の設定を、環境が変わるごとに多数の拡張子に設定するのが大変です
    一括で指定拡張子に設定することができないでしょうか

    いまいち方法が分からないのですが、「スクリーン」右上に
    入力のできない「ファイルの拡張子(X)」という欄があるのですが、
    こちらに拡張子を書くだけで全設定するような感じをイメージしています。

    また昔「ファイルの拡張子(X)」のところになんとか拡張子を入れてた記憶があるのですが
    その時、文字数制限にひっかかるようで、書ききれませんでした。
    制限を広げていただけたらなあと思います。

    デフォルトで java c pl など 言語ごとに 違う設定が入るようになっているようですが
    指定した全拡張子に一括で「自動インデント」「カラー」「正規表現キーワード」などを
    設定したいのです。
    その上で java c pl 等の 追加設定を行うというイメージです

    例えば背景を黒で文字を白にした場合、
    多数ある拡張子ごとに
    それらを1から設定したり「カラー」「正規表現キーワード」の設定ファイル
    を読んだり「スクリーン」を設定しなおしたりする操作が大変です

    また個別に設定した場合に、デフォルトの java c pl 用設定と混ざった設定になるのでしょうか？
