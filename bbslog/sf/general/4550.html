    タグジャンプで開いたファイルからタグジャンプバックするときはそのファイル

    • [4555] Re:要望 タブジャンプバックでファイルを閉じる りーや 2005年07月19日 17:01

      ▼ Modさん
      > ファイルが編集されていたら閉じないとか動作が選べればうれしいです。

      動作を選びたいならキーボードマクロですみます。

      // CloseJumpBack.mac
      S_WinClose(); // ウィンドウを閉じて
      S_TagJumpBack(0); // タグジャンプバック

      これで、編集されていればウインドウ・クローズ時に確認が出るから編集するならキャンセル

      マクロで自動的に判断させるなら JScript マクロになり

      // CloseJumpBack.js
      var stat = Editor.ExpandParameter("${U?更新されている$:更新されていない$}");
      if(stat == "更新されていない") Editor.WinClose(); // ウィンドウを閉じて
      Editor.TagJumpBack(0); // タグジャンプバック

      で、動くみたいです
