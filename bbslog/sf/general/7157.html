      マクロでよければ、下記のように書いて設定すればできると思います。
      Enterで下に移動後、10文字戻るとか直前のスペースまで戻るとかも、書けばできます。
      設定するとき「マクロ」「設定」→OKで閉じる。「キー割り当て」でEnterに割付と２段階必要です。

      // enter_insmode_down.js とかでmacroフォルダに保存
      // Enterにキー割り当て
      if( 1 == Editor.IsInsMode() ){
      // 挿入モードなら改行入力
      Editor.Char( 10 );
      }else{
      // 上書きモードなら下移動
      Editor.Down();
      // 行頭へ移動。必要ならコメントをはずしてください
      // Editor.GoLineTop( 0 ):
      }


      Char( 10 );
      の１行をenter.macとかで保存して、Shift+Enterなどに登録すれば、
      上書きモード中でも改行できるようにもできます。
