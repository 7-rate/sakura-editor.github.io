    • [7266] Re:多重で色分けできますか？ ds14050 2010年11月22日 00:32

      正規表現キーワードを使っても一つのルールにつき一つの色しか指定で
      きないのでお望みの動作はできないと思われます。無理矢理なんとかし
      ようとするなら
      * " から { までを文字列として色分け。
      * { から } までを埋め込み変数として色分け。
      * } から " までを文字列として色分け。
      することとなり、先読みや戻り読みを駆使することで割合は下げられま
      すが、望まない部分まで色分けしてしまう可能性が残ります。

      入れ子になった色分けは、少し前にも要望があがっていたようです。
