    • [2323] RE: マクロ関数　S_FileSaveAs について すい 2002年10月01日 19:48

      S_FileSaveAs ( S1 , i1 , i2 ) ;
      名前を付けて保存

      S1：文字列：保存するファイル名

      i1：整数数値：文字コードセット {
      0：Shift_JIS
      1：JIS
      2：EUC
      3：Unicode
      4：UTF-8
      5：UTF-7
      6：UnicodeBE
      }

      i2：整数数値：改行コード {
      0：変更しない
      1：CRLF
      2：LF
      3：CR
      }

      エラーってどんなエラーです？
      というより、具体的にどういうマクロを書いているのですか？
      マクロそのものを実際に書き込んだ方が話が早そうな気が。
