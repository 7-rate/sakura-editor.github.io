      質問の意味は「複数行をまとめてコメント行にしたい」という事ですか？
      例えば UNIX系のスクリプトの # を複数行にまとめて付けたい、とか
      C++ とかの // を複数行にまとめて付けたい、とか．．．

      それならば複数行（それらの文字を挿入したい位置から）矩形範囲選択してから
      # とか // とかを普通に文字入力するだけです。
