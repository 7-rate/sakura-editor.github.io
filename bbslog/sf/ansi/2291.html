    こんにちは。

    アウトプットウィンドウへ文字列が追加されたとき、
    アンダーラインの表示が残ってしまう問題を修正しました。


    以下のファイルの一行を変更すると修正されるようです。

    ファイル　　　：CEditView_Command.cpp
    メンバー関数　：Command_ADDTAIL
    行番号　　　　：7020

    MoveCursor の三番目の引き数をTRUEに変更。

    変更前　　　　：MoveCursor( nNewPos, nNewLine, FALSE );
    変更後　　　　：MoveCursor( nNewPos, nNewLine, TRUE );


    開発されているみなさん、確認してみてください。
    それでは。

    エディターバージョン ssrc_2002-08-18
    Windows98 + Service Pack1
    MS Visual Studio Ver6.0 + Service Pack5


    ↓ 参考までに再現手順です。

    １．メニューの「ツール」->「外部コマンド実行」と選択し、
    「ファイル名を指定して実行」というダイアログを表示。

    ２．名前に「dir」と入力し、「標準出力を得る」にチェックをいれ、実行。
