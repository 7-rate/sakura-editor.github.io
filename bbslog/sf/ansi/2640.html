  • [2640] 冗長な宣言に対するC++のアウトライン解析について げんた 2003年03月30日 13:48

    class A {
    void func();
    }
    は冗長に
    class A {
    void A::func();
    }
    と記述することもできますが、このように記述するとアウトライン解析で
    Aクラス
    └Aクラス
    　└func(宣言)
    となります。
    Aクラス
    └func(宣言)
    となって欲しいところですけどね。
