    はじめまして。いつも使わせていただいています

    本題ですが、キーワードセットを削除すると、
    タイプ別設定の強調キーワードの設定が
    削除されたキーワードセットより後のセットだけがずれます

    CPropComKeyword.cpp(ssrc_2002-03-07_ver)の312行目あたりに

    for( i = 0; i < MAX_TYPES; ++i ){
    if( nIndex1 < m_Types[i].m_nKeyWordSetIdx ){
    m_Types[i].m_nKeyWordSetIdx--;
    }
    if( nIndex1 < m_Types[i].m_nKeyWordSetIdx2 ){
    m_Types[i].m_nKeyWordSetIdx2--;
    }
    }

    と追加すれば直るみたいです。
    あと、sakura.exe.manifestの文字コードが
    ファイル中にはUFT8と書いてあるのにSJISになっています
