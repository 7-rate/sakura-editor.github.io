  • [27] 行末の空白を削除して改行マクロ ryoji 2003年05月04日 17:52

    行末の空白を削除して改行するマクロを作ってみました。

    // RTrimNL.mac
    // 行末の空白を削除して改行
    S_Char(13); // 文字入力(改行)
    S_MoveHistSet(); // 現在位置を移動履歴に登録
    S_Up(); // カーソル上移動
    S_RTrim(); // 右(末尾)の空白を削除
    S_MoveHistPrev(); // 移動履歴: 前へ

    私はこれを Enter キーに直接割り当てて使っています。
    一般掲示板の「[2614] 空白だけの行の削除」のような要望に合うのではないでしょうか。
    ＃この場合、通常改行は Shift+Enter または、
    ＃S_Char(13)；の一行マクロでやればよいと思います。

      正規表現の定石では
      //キーボードマクロのファイル
      S_ReplaceAll('\\s+\\r\\n', '\\r\\n', 36); //
      S_ReDraw(0); // 再描画
      あたりです、これは全置換になっていますが
      範囲置換でもできます。（DOS改行ですが）

      あとは、ファイル保存時の動作設定で
      行末の空白消去のあるエディタもあります。
