    PPA風で書くと
    function S_ExpandParameter( str1: string ): string;
    というのを追加してはどうでしょうか。

    ■引数
    str1 : 展開される文字列

    ■戻り値
    str1を展開した文字列

    ■解説
    str1で指定した文字列中の特殊文字を展開して返します。
    特殊文字は「外部コマンド実行」で利用できるものと同じです。

    ■利用例
    // このファイルのパス名とカーソル位置を挿入 for JScript
    Editor.InsText( Editor.ExpandParameter( "$F ($x,$y): " ) );

    > 外部コマンドのところの$x、$yのように
    というのも文字列になりますが、取得できるようなりますね。
