    コードの整頓用に、テキストの位置揃えを行うマクロを作成しました。

    http://groups.yahoo.co.jp/group/sakura-editor/files/User/MACRO/
    ⇒TextAlign.js

    int nMoge = 256;
    string strMogeMoge = "もげもげ";
    float fHoge = 0.001;

    ↓ （区切り文字に、= を利用）

    int nMoge = 256;
    string strMogeMoge = "もげもげ";
    float fHoge = 0.001;

    なように、= 等の位置を揃える事が出来ます。
    行末コメントの位置を揃えるのにも使えます。
