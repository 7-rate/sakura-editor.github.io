  • [97] マクロのヘルプ げんた 2001年11月15日 11:46

    キーボードマクロでは記録できないが，マクロファイルを編集することで使えるようになるコマンドがいくつかあります．
    例えば，
    InsText("string") // 文字列挿入
    ExecCommand( "command" ) // コマンド実行
    Up(行数) // 指定行数上に
    GoLineTop(flag) // 1: 空白に関係なく桁位置1に移動，8: 選択して移動

    今後も増えるでしょうからヘルプにも1ページ用意して入れた方が良いと思います．
